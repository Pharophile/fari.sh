# fari.sh — Easy Pharo+Iceberg+Git image setup

_**fari:** To do, to make (eo) — Lighthouses (it)_

**Fari** (re)downloads and prepares a _ready-to-hack [Pharo][] image, so you can
forget about the usual image setup dance: get image, run it, open workspace,
juggle windows, copy-paste, do-it, save image under new name…

```shell
$ git clone git@github.com/$user/$repo.git
$ cd $repo
$ fari.sh
```

See the [Docco](http://ashkenas.com/docco/) generated docs for more information:
https://cdlm.github.io/fari.sh

**Install:** Drop or link `fari.sh` in your `$PATH`.
